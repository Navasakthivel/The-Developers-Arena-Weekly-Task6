# Interactive Sales Dashboard

## Overview

Interactive Sales Dashboard is a Python-based business analytics project designed to analyze sales data and generate meaningful insights through static and interactive visualizations.

The dashboard provides revenue trends, customer segmentation analysis, product performance metrics, channel performance insights, and key business KPIs in a single responsive dashboard.

---

## Features

* Monthly Revenue Trend Analysis
* Product Price Distribution Analysis
* Customer Revenue Distribution
* Interactive Revenue Trend Filtering
* Animated Quarterly Revenue Visualization
* Customer Segmentation Bubble Chart
* Product Performance Sunburst Chart
* Channel Performance Comparison
* KPI Cards for Business Metrics
* Fully Responsive HTML Dashboard

## Installation

Clone the repository:

```bash
git clone <repository-url>
cd project-folder
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## Run the Project

```bash
python dashboard.py
```
